# MyFirstJavaApp (おみくじアプリ) 

Javaの最新バージョンを使って作成した、シンプルなおみくじアプリケーションです。

## 概要
このプロジェクトは、Javaの基礎（配列、ランダム、条件分岐）を学習するために作成しました。
実行すると、「大吉」から「凶」までの運勢がランダムに表示されます。

## 使用技術
- **Language:** Java 25
- **Environment:** macOS (M4 Apple Silicon)
- **IDE:** IntelliJ IDEA

## 実行方法
1. IntelliJ IDEA でプロジェクトを開く
2. `Main.java` を実行する
